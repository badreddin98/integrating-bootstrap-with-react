# React Bootstrap Integration Project with Vite

This project demonstrates the integration of React-Bootstrap components to create a visually appealing and responsive web application. Built with Vite for optimal performance, it includes a HomePage with various Bootstrap components and a NotFound (404) page.

## Features

- Built with Vite for faster development and optimized production builds
- Responsive layout using React-Bootstrap components
- HomePage with hero section, featured products, and call-to-action buttons
- Custom 404 Not Found page with navigation back to home
- Integration of Bootstrap styling and components

## Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

## Development

To start the development server:
```bash
npm run dev
```

The application will be available at [http://localhost:5173](http://localhost:5173).

## Building for Production

To create a production build:
```bash
npm run build
```

To preview the production build:
```bash
npm run preview
```

## Dependencies

- React
- Vite
- React Router DOM
- React-Bootstrap
- Bootstrap

## Project Structure

- `/src/components/` - React components
  - `HomePage.jsx` - Main landing page
  - `NotFound.jsx` - 404 error page
- `/src/assets/` - Images and other static assets
